想必大家已经发现了，会说：“这不就是你首页的那个效果吗？”。嗯,是的，我懒得再写一个JS的文件了......虽然只是改一点变量内容。其实想要实现这个效果还是很简单的，有点js基础，或者有点编程基础的人，看一下文档就可以写出了。这个动画就是在模仿打字，选中和删除这三个动作，我在这里呢就主要说一下思路。

首先，我用了 setInterval(setContent, 4000) 这个命令来实现每间隔 4000 (也就是4秒)就运行一次 setContent 来更换文字。代码如下：

var tv = setInterval(setContent, 4000);
var indexOfId = 0;
var identities = [
   "Airsoft Gamer",
   "Game Commentator",
   "Food Lover",
   "Computer Science Student"
];
var numberOfId = identities.length;
function setContent() {
   if (indexOfId == numberOfId){
      indexOfId = 0;
   }
   display(identities[indexOfId]);
   indexOfId++;
}
